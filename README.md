# AutoCredit Intelligence

AI-powered credit risk scoring and lending decision platform inspired by Moroccan banking practices.

## Features

- Credit risk scoring
- Probability of Default (PD)
- Explainable AI decisions
- Portfolio risk dashboard
- Loan approval recommendations
- Employment and debt analysis

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
